# Acoustic-Tokenisation
An offline fintech protocol using Acoustic FSK to transmit encrypted payments via sound. No Internet/Bluetooth needed.  Features:  Acoustic Engine: 300–3400Hz data waves.  Security: SHA-256 &amp; TOTP verification.  Edge AI: Local fraud risk scoring.  Stack: Python Flask, Web Audio API.  Perfect for network-dead zones. 🇮🇳
